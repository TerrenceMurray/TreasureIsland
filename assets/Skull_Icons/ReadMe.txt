ABOUT THIS PACK
This pack contains a modular set of pixel-art skull assets designed for flexible use across multiple game systems.

Included assets allow you to mix and match skulls, bones, and feather headdresses to create unique icons or progression visuals without needing to draw new art.

Perfect for:
• Progression systems
• Difficulty indicators
• Achievements
• Enemy or boss icons
• Retro-style UI elements
